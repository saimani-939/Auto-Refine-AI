# Auto-Refine AI: Data Cleaning Specialist 
An AI-powered tool that automates data cleaning using Machine Learning.

### Key Features
- **MICE Imputation**: Uses IterativeImputer to predict missing numeric values.
- **Anomaly Detection**: Employs Isolation Forest to identify and flag outliers.
- **LLM Refinement**: Uses Gemini API for fuzzy text standardization.

### How to Run
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/Auto-Refine-AI.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the script: `python main.py`
